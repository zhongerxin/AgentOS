## 自动化机制
1. 文件夹 diff 机制，当这里面的每个文件夹内的文件产生了变更时，都会触发一个事件，事件的内容是变更的文件路径、变更的文件类型、变更的内容等
2. inbox 是外界通知的来源，自动接入我的谷歌邮箱，有新的邮件到达时，就会被同步回一个新的 md 文件，同时会触发一个事件，事件的内容是邮件的元数据、邮件的内容等，这个事件会发送给 agent ，agent 根据邮件内容决定更新 todo、记录日程等
3. 协作空间里的文件夹，其中有一个文件夹是一个插画师外包平台的文件夹，里面的协作方式是，我的 Agent 可以在里面去创建一个外包任务文件夹，这个文件夹里会有一个 md 文档， Agent 会写清楚我的需求，包括任务的描述、任务的截止时间等，插画师的 Agent 会大量的浏览这些不同的文件夹，根据里面的任务要求 md ，来筛选自己能完成的任务，进行完成，然后把产出的插画放到我的文件夹里。当我的文件夹收到这个插画后，会给我的 Agent 发送一条事件，Agent 根据 我的todo、memory、 自动开始在 workspace 工作，并给我产生一条留言回复
