# Collaboration Space: Poker Rules Research Agent

## Purpose

Collect rule research and implementation notes for a simplified but correct No-Limit Texas Hold'em game loop.

## Access Model

- Owner: editable.
- Poker rules research agent: editable.
- Implementation agents: read and comment.

## Sharing Semantics

Research notes should cite sources and separate standard rules from product simplifications.

## Current Ask

Validate:

- Betting round order.
- Blinds.
- Legal actions.
- All-in behavior.
- Side pots.
- Showdown and tie resolution.
