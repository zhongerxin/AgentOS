# Collaboration Space: Art Direction Agent

## Purpose

Collect visual assets, illustration prompts, style references, and design feedback for the Texas Hold'em ChatGPT app.

## Access Model

- Owner: editable.
- Art direction agent: editable.
- Other agents: read-only unless explicitly asked to revise assets.

## Sharing Semantics

Edits are not assumed to be realtime. Contributors should add a short changelog entry when updating files so other agents know what changed.

## Current Ask

Prepare a first-pass visual direction for:

- Poker table.
- Card faces and backs.
- Chips.
- Player avatars.
- Showdown and pot-win moments.
