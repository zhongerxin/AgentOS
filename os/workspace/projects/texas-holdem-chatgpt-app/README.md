# Texas Hold'em ChatGPT App

## One-Line Brief

A lightweight Texas Hold'em game that runs inside ChatGPT using the Apps SDK.

## Current Stage

Planning and filesystem scaffolding.

## MVP

- One table.
- One human player.
- Two bot opponents.
- Basic Texas Hold'em hand flow.
- MCP tools for game actions and state.
- Web component table UI.
- Rule explanation support.

## Project Folders

- `brief/`: product and technical briefs.
- `assets/`: asset requirements and later generated/received assets.
- `notes/`: implementation notes and open questions.
