# Wiki

This is the compiled knowledge base for Project One.

The intended workflow follows the LLM wiki pattern:

1. Keep raw external or incoming material under `raw-sources/`.
2. Compile useful conclusions into topic pages.
3. Treat compiled pages as the current working truth.
4. Update wiki pages when decisions change.
5. Do not rely on chat history as the durable memory layer.

## Topic Map

- `apps-sdk/`: ChatGPT Apps SDK implementation knowledge.
- `poker/`: Texas Hold'em rules and game design notes.
- `product/`: MVP scope and user experience notes.
- `chrome-extension/`: HighlightGPT monitoring and growth notes.
- `agent-os/`: Design of this local filesystem OS.
