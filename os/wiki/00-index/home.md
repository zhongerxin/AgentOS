# Project One Wiki Home

## Active Question

How should a solo developer build a focused ChatGPT Apps SDK Texas Hold'em game while coordinating help from agents and collaborators through a local filesystem OS?

## Current Working Truth

- The first project is a Texas Hold'em game for ChatGPT.
- The app should start with a small playable loop, not a full poker platform.
- Apps SDK work requires an MCP server and can optionally include a web component rendered in ChatGPT.
- Collaboration assets and research should land in `collaboration/` first, then be promoted into the wiki.
- Memory should store durable decisions and recurring observations, not every raw message.
